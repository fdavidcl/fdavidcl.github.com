Truco (Javascript): Gestionando las clases de los elementos

Un poco conocido atributo de elementos HTML en Javascript, classList, nos permite añadir o quitar clases CSS rápida y fácilmente:

var mi_elemento = document.getElementById("elem_html");

mi_elemento.classList.add("nueva_clase");
mi_elemento.classList.remove("otra_clase");
// No importa si el elemento tiene o no la clase 'otra_clase',
// se quitará si la tiene
